WhatzCookin is a mobile-responsive web app that takes an ingredient and returns a recipe suggestion based on a userâ€™s preferences. WhatzCookin utilizes a taste algorithm based on the flavors (sweet, spicy, bitter, etc.) of a user's favorited and rated recipes to create a user's taste profile. Bomber Math is a multiplication table game where you face off against an opponent to see who can answer a set of randomly generated problems quickest. The project was a top 100 finisher for the Hack Summit 2016 hackathon. It is built using Node.js, Express.js, and socket.io to create a quick, responsive web app that handles multiple simultaneous players.

Rails Online Shop is an online shop built with Ruby on Rails. A user can login and shop by placing items in their cart. Some key features include the ability to filter shop items based on their location or status. There is also an entirely separate admin dashboard for the 'shop owner' to access and update at will.

Snow Days is a web app built with Sinatra. A user logs in and favorites ski resorts they love. When that resort gets a new dusting of snow, the user is alerted via text message. Snow Days utilizes the Weather Underground API, Twillio API, and Google OAuth.
