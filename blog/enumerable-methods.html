Enumerable#cycle
10-06-2015


To call cycle, use the below format:

enum.cycle(n){|object| block}

Where enum is some collection, n is some positive integer or nil, and block is some block of code.

#cycle calls the block for each element of enum n times. If n is nil or left out (e.g. enum.cycle {|object| block}) then #cycle calls block for each element of enum forever. If enum is empty then #cycle does nothing. If a non-positive n is given then #cycle does nothing. #cycle returns nil whenever the loop finishes (if it finishes without being interrupted). If there is no block given (e.g. enum.cycle(n)), an enumerator is returned.
